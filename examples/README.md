# Example Generated Repository

This directory shows what your GitHub repository will look like after the extension syncs your LeetCode activity.

## Repository Structure

```
leetcode-streak/
├── README.md                    # Auto-generated repository description
└── streak/                      # Daily activity files
    ├── 2025-01-15.md
    ├── 2025-01-16.md
    ├── 2025-01-17.md
    └── ...
```

## Example Commit History

```
* Solved: Two Sum - 2025-01-17
* LeetCode daily activity - 2025-01-16
* Solved: Add Two Numbers - 2025-01-15
* Initialize LeetCode streak tracker
```

## What Each File Contains

Each daily file in the `streak/` directory contains:
- Date and time of activity
- Problem name (if detected)
- Timestamp
- Automated commit message

## GitHub Contribution Graph

Your GitHub profile will show:
- Green squares for each day you solve problems
- Maintained contribution streak
- Activity visible to employers and other developers
