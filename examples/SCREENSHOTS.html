Extension Popup

The extension features a beautiful gradient popup interface where users can configure their GitHub credentials and monitor their streak.

Key Features

  • Automatic GitHub commit creation when solving LeetCode problems
  • Secure token storage using Chrome's sync storage
  • Beautiful gradient UI with real-time status updates
  • Daily streak tracking and statistics
  • Browser notifications for sync confirmations
  • Automatic repository creation if it doesn't exist
  • One commit per day to maintain clean contribution graph
  • Hourly background checks for activity

Quick Setup Guide

  1. Install the extension in Chrome (or any Chromium-based browser)
  2. Create a GitHub Personal Access Token with 'repo' scope
  3. Click the extension icon and enter your credentials
  4. Start solving LeetCode problems!
  5. Watch your GitHub contribution graph grow automatically

How It Works

The extension uses three main components:

  • Content Script: Monitors LeetCode pages for solved problems
  • Background Worker: Handles GitHub API calls and creates commits
  • Popup Interface: Provides configuration and status display
