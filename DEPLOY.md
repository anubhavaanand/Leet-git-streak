## Push to GitHub

```bash
git push origin main
```

## Enable GitHub Pages

After pushing, enable GitHub Pages:

1. Go to your repository: https://github.com/anubhavaanand/Leet-git-streak
2. Click **Settings** â†’ **Pages** (in left sidebar)
3. Under **Source**, select:
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. Click **Save**
5. Wait a few minutes for GitHub to build and deploy
6. Your site will be available at: `https://anubhavaanand.github.io/Leet-git-streak/`

## Alternative: Manual GitHub Pages Setup

If the workflow doesn't work automatically:

1. Go to repository **Settings** â†’ **Pages**
2. Select **Source**: `Deploy from a branch`
3. Select **Branch**: `main` / `root`
4. Click **Save**

## Verify Deployment

- Visit: https://anubhavaanand.github.io/Leet-git-streak/
- You should see the beautiful landing page with installation instructions!

## Installing the Extension from GitHub

Users can now:
1. Visit your GitHub Pages site
2. Click "Download from GitHub" 
3. Clone or download the repository
4. Load it in Brave/Chrome/Edge as an unpacked extension

---

**Note**: The extension itself runs in the browser, but the GitHub Pages site serves as a landing page and installation guide!
