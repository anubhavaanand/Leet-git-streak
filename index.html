For Brave Browser (Linux)

  1. Open Brave browser
  2. Navigate to brave://extensions/
  3. Enable "Developer mode" (toggle in top-right)
  4. Click "Load unpacked"
  5. Select the folder containing this extension
  6. The extension should now appear in your extensions list!

Install from GitHub Repository

  1. Clone or download this repository:
    git clone https://github.com/anubhavaanand/Leet-git-streak.git cd Leet-git-streak
  2. Follow the Brave installation steps above
  3. Or download as ZIP from GitHub and extract it
